# 🦜 앤디 장난감 사주기

앤디에게 장난감을 사주고 성장시키는 간단한 클리커 게임입니다.

## 🎮 플레이 방법

* **돈 벌기** 버튼을 눌러 돈을 획득합니다.
* **업그레이드**를 구매하여 클릭당 획득하는 돈을 증가시킬 수 있습니다.
* **장난감 구매**를 통해 앤디의 호감도를 올리고 자동 수익을 획득할 수 있습니다.
* 진행 상황은 **저장** 버튼으로 저장할 수 있습니다.

## 📈 성장 요소

### 업그레이드

클릭당 획득하는 돈이 증가합니다.

### 장난감

앤디가 새로운 반응을 보이며 자동 수익이 증가합니다.

## 💾 저장

게임 데이터는 브라우저의 LocalStorage를 이용해 저장됩니다.

저장되는 데이터:

* 현재 돈
* 클릭당 수익
* 자동 수익
* 업그레이드 비용
* 장난감 비용
* 앤디 레벨

## 🛠 사용 기술

* HTML
* CSS
* JavaScript
* LocalStorage

## 📷 앤디의 변화

* Lv.1 : 앤디가 관심을 가진다...
* Lv.5 : 앤디는 리본을 가지고 놀고 있다...
* Lv.10 : 앤디는 너무 즐거워 휴식을 취하고 싶다...

## 🚀 실행

저장소를 클론한 뒤 `index.html`을 실행하거나 GitHub Pages를 통해 플레이할 수 있습니다.

즐겁게 플레이하세요!
